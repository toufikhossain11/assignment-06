# 🚀 Digitools Platform

## 📌 Project Name

**Digitools Platform**

---

## 📝 Description

Digitools Platform is a modern SaaS-based web application where users can explore various digital tools and purchase them easily. The platform includes product listing, dynamic cart functionality, and a smooth user experience with responsive design. Users can add products to the cart, remove them, and proceed to checkout with real-time updates and notifications.

---

## 💻 Technologies Used

* React.js
* Tailwind CSS
* React Icons
* React Toastify
* JavaScript (ES6)
* JSON Data Handling

---

## ✨ Key Features

### 1️⃣ Dynamic Product & Cart System

Users can browse products and add them to the cart. The cart updates instantly with total product count and allows removing individual items or clearing all items with checkout.

### 2️⃣ Toggle System (Cart & Products)

A smart toggle system allows users to switch between the product section and the cart section. By default, the product section is visible, and the cart shows an empty message initially.

### 3️⃣ Interactive UI with Notifications

React Toastify is used to show real-time alerts for actions like adding to cart, removing items, and proceeding to checkout, improving user experience.

---

## 📱 Responsive Design

The entire application is fully responsive and optimized for mobile, tablet, and desktop devices.
